## 🎨 Color System

```
BACKGROUNDS:
  ┌─────────────┐
  │ #FFFFFF     │ Light sections (About, Prepare, Booking)
  │ (White)     │
  └─────────────┘

  ┌─────────────┐
  │ #000000     │ Dark sections (Details, FAQ, Footer)
  │ (Black)     │
  └─────────────┘

ACCENTS:
  ┌─────────────────────────────────┐
  │ oklch(0.75 0.2 70) - GOLD       │ Luxury accent color
  │ Hover states, borders, highlights
  └─────────────────────────────────┘

TEXT:
  ┌──────────────────┐
  │ #000000 (Black)  │ On light backgrounds
  │ oklch(0.145 0 0) │
  └──────────────────┘

  ┌──────────────────────┐
  │ #FFFFFF (White)      │ On dark backgrounds
  │ oklch(0.985 0 0)     │
  └──────────────────────┘
```

## 📐 Layout Breakpoints

```
Mobile:   0px     - 768px (single column, full width)
Tablet:   768px+  - two column layouts
Desktop:  1024px+ - optimal spacing with max-width container
```

## ✨ Interactive Elements

### Buttons

```
Primary Button (Gold):
  Background:     Gold (oklch(0.75 0.2 70))
  Hover:          Darker gold + shadow
  Transition:     0.3s ease

Secondary Button (Border):
  Border:         Gold
  Hover:          Gold background + black text
  Transition:     0.3s ease
```

### Cards (Details Section)

```
Default State:
  Border:         rgba(255,255,255,0.1)
  Hover:
    - Border:     rgba(255,215,0,0.5) [Gold with opacity]
    - Icon:       Scale 1.1
    - Transition: 0.3s
```

### Accordion (FAQ)

```
Closed:
  Indicator:      "+"
  Background:     Transparent
  Hover:          Subtle bg-white/5

Open:
  Indicator:      "−"
  Background:     bg-white/5
  Content:        Smooth expand animation
```

## 📱 Responsive Typography

```
Mobile (0px-768px):
  H1: 36px / 1.5 leading
  H2: 28px / 1.4 leading
  Body: 16px / 1.6 leading

Desktop (768px+):
  H1: 56px / 1.2 leading (hero) → 48px (other sections)
  H2: 40px / 1.3 leading
  Body: 18px / 1.7 leading
```

All with:

- Font weight: 300 (light) for elegance
- Letter spacing: wide (`tracking-wider`, `tracking-widest`)
- Font family: Geist (modern, clean, premium)
